go directive

go mod 里面的 go 指令。

go 指令表示一个模块是按照特定版本的 Go 语义编写的。版本必须是有效的 Go 版本，例如 1.9、1.14 或 1.21rc1。

go 指令设置使用此模块所需的最低 Go 版本。在 Go 1.21 之前，这个指令仅是建议性的；但现在它是一个强制性要求：Go 工具链拒绝使用声明了更新 Go 版本的模块。

go 指令是选择运行哪个 Go 工具链的输入之一。详见Go 工具链。
